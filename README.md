# OTS-DevOps

Infrastructure and DevSecOps automation for the Online Testing System.

## Project Goals

- Provision AWS infrastructure using Terraform
- Configure servers using Ansible
- Deploy the Online Testing System with Docker
- Build CI/CD using Jenkins
- Integrate security scanning:
  - Bandit
  - Gitleaks
  - Trivy
  - SonarQube
- Monitor the platform using:
  - Prometheus
  - Grafana
  - Loki
- Implement zero-downtime deployments
